Project Overview

Developed an interactive Netflix Content Performance Analytics dashboard in Power BI to analyze viewing patterns, content distribution, and audience preferences across genres, regions, and release years.

Key Features Implemented:

▸Integrated datasets covering titles, genres, release years, ratings, and regional availability

▸Created 13+ interactive Power BI visuals with slicers for genre, country, release year, and content type

▸Built DAX measures to track content distribution, average ratings, and year-over-year trends

▸Designed drill-through pages for title-level and genre-level performance analysis

▸Delivered an executive-style summary page highlighting top genres, content growth trends, and regional insights

Challenges & Solutions

⚡
Challenge

Netflix content data was available in raw datasets but lacked structure and interactivity, making it difficult to analyze content trends, genre performance, and regional distribution through static reports.

✦
Solution

Developed a centralized Power BI dashboard using Power Query for data cleaning and transformation. Implemented a star-schema data model and optimized DAX measures to enable dynamic filtering, drill-down analysis, and clear storytelling through visuals.

Results & Outcomes

Reduced manual analysis effort by 70%
Enabled fast exploration of content trends by genre, year, and region
Identified high-performing genres and growth in original content
Improved visibility into platform-wide content distribution and trends
